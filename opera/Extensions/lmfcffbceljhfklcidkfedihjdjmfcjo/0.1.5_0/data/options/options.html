Website Theme Manager enhances your browsing experience by allowing you to customize the way a website looks. By using custom styles (css) you can change colors, fonts, and almost all HTML contents within a page; meaning completely redesigning the entire page. Website Theme Manager enables you to easily manage (add, edit or remove) styles. In order to add styles, there are two options available. First, you can visit userstyle website, find your desired theme and click on "Add This Theme", the Theme will be added to the table automatically. Second option is to copy and paste your desired theme (css) and add it to the theme table here (an example is provided below). Once a theme is added, you can enable or disable the theme or even permanently delete it from memory. Note: in order to report bugs please visit Website Theme Manager's homepage and fill the bug report form.

Example: In order to add themes manually, a sample style (Theme) is shown below. Column (1) shows the theme number. Column (2) is where you put the Theme Name. Theme name is used to find themes from the memory, to add a name, please use small letters with no space(s). Column (3) is the author name. Column (4) is the CSS (style) code. You can use your own style or styles from userstyles collection. Column (5) is to activate or inactivate the theme, blue color indicates an active and gray color indicates an inactive code (clicking the button will swap the colors). Column (6) is to permanently delete a theme item. To add themes automatically, as mentioned above, please visit userstyle website, find your desired theme and click on "Add This Theme".
